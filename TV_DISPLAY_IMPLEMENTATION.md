## 📱 使用步骤

### 在电视机上设置

1. **打开浏览器**
   - 推荐使用 Chrome 或 Edge
   - 输入 URL：`http://你的域名/tv-display`

2. **登录管理员账号**
   - 使用管理员账号登录
   - 确保有管理员权限

3. **进入全屏模式**
   - 页面会自动尝试进入全屏
   - 如果失败，按 `F11` 手动全屏

4. **固定页面**
   - 将浏览器设置为"固定标签页"
   - 设置浏览器为"启动时打开此页面"
   - 禁用浏览器自动休眠

5. **优化显示**
   - 调整电视机亮度（适合长时间观看）
   - 确保网络连接稳定
   - 关闭浏览器扩展（提升性能）

## 🐛 故障排除

### 问题 1：无法访问页面
- **原因**：不是管理员账号
- **解决**：使用管理员账号登录

### 问题 2：数据不更新
- **原因**：网络连接问题或浏览器缓存
- **解决**：检查网络，清除浏览器缓存，强制刷新（Ctrl+F5）

### 问题 3：全屏模式不工作
- **原因**：浏览器不支持全屏 API
- **解决**：按 `F11` 手动全屏

### 问题 4：字体太小
- **原因**：浏览器缩放设置
- **解决**：确保浏览器缩放为 100%，或调整电视机显示设置
