# 📺 TV Display 页面使用说明

## 🎯 功能概述

TV Display 页面融合了 **Top 10 排行榜滚动展示** 和 **亮点数据展示**，专为球房电视机设计。

---

## 🚀 访问方式

### 方式 1：直接访问 URL
```
http://你的域名/tv-display
```

### 方式 2：本地测试
```
http://localhost:3001/tv-display
```

---

## ✨ 主要功能

### 1. **Top 10 排行榜展示**（左侧）
- 🥇 🥈 🥉 Top 3 大卡片展示（带排名徽章）
- 4-10 名选手自动滚动展示
- 显示数据：
  - 排名
  - 选手名称
  - 当前积分
  - W/L 记录
  - Win Rate
  - Break & Run 次数

### 2. **亮点数据展示**（右侧）
- 🏆 **Best Performance**：最佳表现选手
- 🎯 **Most Break & Run**：最多 B&R 选手
- 📈 **Fastest Riser**：最快上升选手
- ⚡ **Winning Streak**：连胜记录
- 📅 **Most Active**：最活跃选手
- 💯 **Highest Win Rate**：最高胜率

### 3. **自动功能**
- ✅ 自动刷新数据（每 60 秒）
- ✅ 自动滚动排行榜（每 5 秒）
- ✅ 自动切换组别（Pro/Student，每 2 分钟）
- ✅ 自动全屏模式（打开页面时）
- ✅ 实时时间显示

### 4. **底部统计**
- 👥 总选手数
- 🎯 总比赛场次
- 🏆 总赛事数
- 🎯 总 Break & Run 次数
- 最后更新时间

## 🎯 使用步骤

### 1. 在电视机上打开浏览器
- 推荐使用 Chrome 或 Edge
- 输入 URL：`http://你的域名/tv-display`

### 2. 进入全屏模式
- 页面会自动尝试进入全屏
- 如果失败，按 `F11` 手动全屏

### 3. 设置浏览器
- 关闭浏览器工具栏
- 设置浏览器为"全屏模式"
- 禁用自动休眠

### 4. 固定页面
- 将浏览器设置为"固定标签页"
- 设置浏览器为"启动时打开此页面"

---

## 💡 优化建议

### 1. **网络连接**
- 确保电视机有稳定的网络连接
- 建议使用有线网络

### 2. **浏览器设置**
- 禁用浏览器扩展
- 关闭自动更新
- 设置浏览器为"无痕模式"（可选）

### 3. **显示设置**
- 调整电视机亮度（适合长时间观看）
- 设置电视机为"自动关闭"（可选）

### 4. **性能优化**
- 关闭其他标签页
- 定期清理浏览器缓存
- 重启浏览器（每周一次）

---

## 🐛 故障排除

### 问题 1：页面不显示数据
- **原因**：网络连接问题
- **解决**：检查网络连接，刷新页面

### 问题 2：数据不更新
- **原因**：浏览器缓存
- **解决**：清除浏览器缓存，强制刷新（Ctrl+F5）

### 问题 3：全屏模式不工作
- **原因**：浏览器不支持全屏 API
- **解决**：按 `F11` 手动全屏

### 问题 4：动画卡顿
- **原因**：电视机性能不足
- **解决**：降低动画效果，或使用性能更好的设备

## 📊 数据说明

### 数据来源
- **排行榜数据**：来自 `public_users` 表
- **积分数据**：来自 `ranking_point_history` 表
- **统计数据**：来自 `user_year_stats` 表

### 数据更新
- **自动更新**：每 60 秒刷新一次
- **手动更新**：刷新页面（F5）

### 数据范围
- **当前年份**：2026 年数据
- **组别**：Pro Division 或 Student Division
- **显示数量**：Top 10 选手
