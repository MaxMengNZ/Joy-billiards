## 🎨 设计细节

### 颜色方案
- **背景**：`#0a0a0a` (深黑)
- **卡片背景**：`#1a1a2e` (深蓝灰)
- **主色**：`#FFD700` (金色) / `#FF6B35` (橙色)
- **文字**：`#FFFFFF` (白色) / `#E0E0E0` (浅灰)
- **强调**：`#00FF88` (绿色成功) / `#FF4444` (红色警告)

### 字体
- **标题**：Bold 96px
- **数据**：Bold 64px
- **标签**：Regular 32px
- **小字**：Regular 24px

### 动画
- **滚动速度**：每 3-5 秒切换
- **淡入淡出**：0.5 秒过渡
- **数字滚动**：1 秒动画
- **高亮闪烁**：2 秒循环

---

## 📐 响应式设计

### 横屏模式（推荐）
- 16:9 或 21:9 比例
- 左右分栏布局
- Top 3 横向排列

### 竖屏模式（备选）
- 9:16 比例
- 上下滚动布局
- Top 3 纵向排列

---

## ⚙️ 技术规格

### 性能要求
- **刷新频率**：30-60 秒自动刷新
- **动画流畅度**：60 FPS
- **数据加载**：< 2 秒

### 浏览器兼容
- Chrome/Edge（推荐）
- 全屏 API 支持
- 自动刷新支持

## 🎯 实现优先级

### Phase 1（核心功能）
1. ✅ 基础全屏页面
2. ✅ Top 10 排行榜展示
3. ✅ 自动刷新数据
4. ✅ 基础滚动动画

### Phase 2（增强体验）
1. ✅ 多模式轮播
2. ✅ 亮点数据展示
3. ✅ 实时比赛动态
4. ✅ 统计数据总览

### Phase 3（高级功能）
1. ✅ 管理员配置面板
2. ✅ 自定义刷新间隔
3. ✅ 模式选择
4. ✅ 主题切换

## 💡 额外功能建议

1. **QR Code 显示**
   - 在屏幕上显示二维码
   - 顾客扫码查看完整排行榜

2. **音效提示**（可选）
   - 新数据更新时的提示音
   - 模式切换音效

3. **多语言支持**
   - 中英文切换
   - 根据球房需求

4. **移动端适配**
   - 手机也可以查看
   - 但主要针对电视大屏

---

## ❓ 需要确认的信息

1. **电视机尺寸**：多少寸？分辨率？
2. **展示位置**：横屏还是竖屏？
3. **刷新频率**：希望多久更新一次？
4. **重点内容**：最想突出展示哪些数据？
5. **访问方式**：通过 URL 直接访问，还是需要登录？
6. **网络环境**：电视机如何连接网络？
