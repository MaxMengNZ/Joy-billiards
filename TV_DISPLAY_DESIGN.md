# 📺 球房电视机实时数据展示设计方案

## 🎯 设计目标
在球房电视机上实时滚动展示亮眼的球员数据，吸引顾客注意，提升球房氛围。

---

## 📋 展示内容设计

### 1. **主展示区域（轮播模式）**

#### 模式 A：排行榜 Top 10 滚动展示
- **布局**：横向滚动，每屏显示 3-5 名选手
- **内容**：
  - 排名徽章（🥇 🥈 🥉）
  - 选手头像/名称
  - 当前积分
  - W/L 记录
  - Win Rate
  - Break & Run 次数
- **动画**：从左到右平滑滚动，每 5 秒切换一组

#### 模式 B：亮点数据展示
- **内容**：
  - 🏆 本周最佳表现（最高 Win Rate）
  - 🎯 最多 Break & Run
  - 📈 积分增长最快
  - ⚡ 连胜记录
  - 🌟 新星选手（最近加入且表现突出）
- **动画**：卡片式展示，淡入淡出切换

#### 模式 C：实时比赛动态
- **内容**：
  - 正在进行的比赛
  - 最近完成的比赛结果
  - 即将开始的比赛预告
- **动画**：实时更新，闪烁提示新数据

#### 模式 D：统计数据总览
- **内容**：
  - 总参赛人数
  - 总比赛场次
  - 总 Break & Run 次数
  - 本周活跃选手数
- **动画**：数字滚动增长效果

## 🎨 视觉设计

### 配色方案
- **背景**：深色渐变（#1a1a2e → #16213e）或纯黑（#000000）
- **主色**：金色/橙色高亮（#FFD700 / #FF6B35）
- **文字**：白色/浅灰色（#FFFFFF / #E0E0E0）
- **强调色**：绿色（成功）、红色（警告）、蓝色（信息）

### 字体大小
- **标题**：72-96px（大屏幕易读）
- **数据**：48-64px
- **标签**：32-40px
- **小字**：24-28px

## ⚙️ 技术实现方案

### 1. **创建新页面：`/tv-display`**
- 全屏模式（无导航栏、无滚动条）
- 自动刷新数据（每 30 秒）
- 响应式设计（适配横屏电视）

### 2. **功能特性**
- ✅ 自动轮播多个展示模式
- ✅ 平滑滚动动画
- ✅ 实时数据更新
- ✅ 可配置刷新间隔
- ✅ 全屏 API 支持
- ✅ 暗色主题（适合长时间观看）

### 3. **数据展示优先级**
1. **Pro Division** 和 **Student Division** 分别展示
2. 可按当前年份或全部数据展示
3. 突出显示 Top 3
4. 显示最近活跃的选手

---

## 🎬 动画效果

### 滚动动画
- **排行榜滚动**：向上滚动，每 3 秒切换一条
- **模式切换**：淡入淡出，3 秒过渡
- **数据更新**：数字滚动增长效果
- **高亮效果**：Top 3 有发光动画

### 视觉特效
- **排名徽章**：轻微旋转/闪烁
- **积分数字**：滚动增长动画
- **新数据提示**：闪烁/高亮 2 秒

## 📱 页面结构

```
TV Display Page
├── Header (固定)
│   ├── Logo
│   ├── 标题 "Joy Billiards Leaderboard"
│   └── 当前时间
│
├── Main Content (轮播区域)
│   ├── Mode A: Top 10 排行榜滚动
│   ├── Mode B: 亮点数据展示
│   ├── Mode C: 实时比赛动态
│   └── Mode D: 统计数据总览
│
└── Footer (固定)
    ├── 总统计数字
    └── 刷新时间
```

## 🔧 配置选项（管理员可设置）

1. **刷新间隔**：30秒 / 60秒 / 120秒
2. **展示模式**：选择要展示的模式（可多选）
3. **显示组别**：Pro / Student / 全部
4. **显示年份**：当前年份 / 全部年份
5. **自动全屏**：打开页面自动进入全屏模式
